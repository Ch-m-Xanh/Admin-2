Thông tin hệ thống

Kiến trúc: Node.js + Express + MongoDB (Mongoose) + Socket.IO
Vai trò: Backend trung tâm phục vụ App Mobile Chạm Xanh và trang quản trị này
Realtime: Mỗi khi bạn thêm/sửa/xoá bài viết hoặc cây, app đang mở sẽ tự cập nhật.
API base:

Kết nối App Mobile

Trong file Mobile/.env, đặt:
EXPO_PUBLIC_API_BASE_URL=<địa-chỉ-server>/api
Ví dụ khi deploy: https://cham-xanh-admin.onrender.com/api
